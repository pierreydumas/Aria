# E8-S86 — engine_focus Wiring: Focus Level → work_cycle Adaptive Routing

**Sprint:** E8  
**Type:** Feature / Code  
**Priority:** P1 — core focus system integration; unlocks token savings at L1/L2  
**Effort:** M  
**Depends on:** S-78 (HEARTBEAT docs), S-71 (engine_focus.py CRUD already done)  

## Problem

`engine_focus.py` and `FocusProfileEntry` (E7 S-70/S-71) provide:
- 8 seeded focus profiles with `depth_level` (L1/L2/L3), `model_tier`, `max_skills`, `roundtable_eligible`
- Full CRUD API at `GET/POST /api/engine/focus`
- Active profile tracking

But nothing wires the active focus profile into actual runtime behavior. Every work_cycle still runs at the same depth regardless of focus level.

## Work To Do

### 1. aria_engine/heartbeat.py (or equivalent cron runner)

Add focus level check at start of each `work_cycle`:

```python
async def get_active_focus_level() -> str:
    """Returns 'L1', 'L2', or 'L3'. Defaults to 'L2' on error."""
    try:
        resp = await api_client.get_memory(key="active_focus_level")
        level = resp.get("value", "L2")
        return level if level in ("L1", "L2", "L3") else "L2"
    except Exception:
        return "L2"  # safe default

FOCUS_ROUTING = {
```

### 2. work_cycle execution path

```python
async def work_cycle():
    focus_level = await get_active_focus_level()
    config = FOCUS_ROUTING[focus_level]
    
    # Check CB first (existing logic)
    if circuit_breaker_open():
        write_degraded_log("work_cycle")
        return
    
    # Scale goal fetch by focus level
    goals = await api_client.get_goals(status="active", limit=config["max_goals"])
    
    # L1: single direct action, no delegation
    if focus_level == "L1":
        await do_single_action(goals[0], config)
        return
    
    # L2: delegate if task > 5 min estimate
    if focus_level == "L2":
        task = select_best_goal(goals)
        if is_complex(task):
            await delegate_to_agent(task, max_concurrent=2)
        else:
            await do_single_action(task, config)
        return
    
    # L3: full depth, roundtable eligible for complex multi-domain tasks
    if focus_level == "L3":
        task = select_best_goal(goals)
        if is_multi_domain(task) and config["roundtable"]:
            await trigger_roundtable(task, agents=["analyst", "creator", "devops"])
        elif is_complex(task):
            await delegate_to_agent(task, max_concurrent=5)
        else:
            await do_single_action(task, config)
```

### 3. six_hour_review — upgrade at L3

When `focus_level == "L3"` and cron job is `six_hour_review`:
