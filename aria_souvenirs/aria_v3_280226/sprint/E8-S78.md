# E8-S78 — HEARTBEAT: Focus Level Check + L1/L2/L3 Routing

**Sprint:** E8  
**Type:** Documentation / Behavior  
**Priority:** P1 — blocks all other focus-aware work  
**Effort:** M  

## Problem

`engine_focus.py` and `FocusProfileEntry` were built in E7 (S-70/S-71) with 8 seeded focus profiles including L1/L2/L3 depth levels. However:

- HEARTBEAT.md `work_cycle` does NOT check the active focus profile at wakeup
- No behavioral guidance tells Aria how to scale delegation based on focus level
- Every work_cycle runs at full depth regardless of focus — wasting tokens on L1 tasks

## Focus Level Definitions

| Level | Depth | Model | Sub-agents | Roundtable | Max Skills |
|-------|-------|-------|-----------|-----------|-----------|
| L1 (surface) | Shallow | qwen3-mlx (local) | NO | NO | 1–2 |
| L2 (medium) | Standard | kimi (free cloud) | YES (max 2) | NO | 3–4 |
| L3 (deep) | Full | kimi / trinity-free | YES (max 5) | YES | unlimited |

## Work To Do

### 1. HEARTBEAT.md — add focus level check at start of `work_cycle`

Insert after the circuit breaker check, before goal selection:

```markdown
### 0. Check Active Focus Level (NEW)
```tool
aria-api-client.get_memory({"key": "active_focus_level"})
```
- If missing or error: default to **L2**
- If **L1**: use local model, NO sub-agents, max 2 skills, single api_client call
- If **L2**: standard model, up to 2 sub-agents, normal delegation
- If **L3**: full depth, roundtable eligible, all skills available
```

### 2. HEARTBEAT.md — add L1/L2/L3 routing table to Standing Orders

```markdown
## Focus Level Routing

| Level | Work Cycle Behavior |
|-------|---------------------|
| L1 | health_check_all → get_goals(limit=1) → ONE action → log → STOP |
| L2 | health_check → get_goals(limit=3) → ONE action → delegate if > 5min task |
| L3 | health_check → get_goals(limit=5) → full analysis → roundtable if complex |
```

### 3. Add focus level set/clear commands to cheatsheet

```tool
# Set focus level
aria-api-client.set_memory({"key": "active_focus_level", "value": "L1"})
# Clear (return to L2 default)
aria-api-client.delete_memory({"key": "active_focus_level"})
```
